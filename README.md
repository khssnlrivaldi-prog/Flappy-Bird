# 🐦 Flappy Bird Enggang

Game Flappy Bird bertema Burung Enggang 

## 🎮 Fitur
- Burung Enggang sebagai karakter utama
- AI Play
- Pause / Resume (Tombol P)
- High Score
- Medal System
- Sound Effect
- Animasi Sayap Burung

## 🛠️ Teknologi
- HTML5
- CSS3
- JavaScript
- Canvas API
- Node.js (Express)

## ▶️ Menjalankan Project

Install dependency

```bash
npm install
```

Jalankan server

```bash
npm start
```

atau

```bash
node server.js
```

Buka browser

```
http://localhost:3000
```
